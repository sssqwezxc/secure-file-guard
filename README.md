# 🔐 Secure File Guard

## 📌 Описание проекта

**Secure File Guard** — это десктопное приложение с графическим интерфейсом, предназначенное для шифрования и расшифровки файлов с использованием пароля.

Программа позволяет пользователю защитить свои файлы от несанкционированного доступа, а также восстановить их при помощи правильного пароля.

---

## 🎯 Актуальность

В современном цифровом мире защита информации является одной из ключевых задач. Пользователи хранят на своих устройствах личные данные, документы, фотографии и другую важную информацию.

Одной из самых распространённых угроз является несанкционированный доступ к файлам. Данный проект демонстрирует простой способ защиты информации с помощью шифрования.

## 🔐 Принцип работы

Программа использует:

* хеширование пароля с помощью SHA-256
* генерацию случайной соли
* XOR-шифрование содержимого файла

При шифровании файл преобразуется в зашифрованный набор байтов, который невозможно прочитать без правильного пароля. Для восстановления исходного содержимого необходимо выполнить расшифровку с тем же паролем.
